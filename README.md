### AI Coach
Работает через Anthropic Claude API. Анализирует текущее состояние доски и даёт персональный совет в стиле тренера — называет конкретную ячейку и объясняет логику (Naked Singles, Hidden Singles, Pointing Pairs и др.), не раскрывая ответ напрямую.

### Daily Challenge
Пазл генерируется детерминированно на основе текущей даты (seeded RNG) — значит, все игроки мира решают одну и ту же задачу в один и тот же день.

### Монетизация
Заложена кнопка **"Upgrade — $2.99/mo"** с Pro-фичами:
- Безлимитные подсказки
- Кастомные темы (Midnight, Ocean, Forest)
- AI Coach без API ключа
- Продвинутая статистика и стрики

## Технический стек

- **Vanilla JavaScript** — нет зависимостей, нет билда, работает из коробки
- **Чистый CSS** с CSS-переменными (light/dark тема)
- **Алгоритм генерации** — backtracking solver с seeded RNG (mulberry32)
- **AI** — Anthropic Claude API (`/v1/messages`)
- **Хранилище** — `localStorage` для лидерборда и настроек темы
- **Деплой** — GitHub Pages (один файл `index.html`)

## Почему это ценно

Существующие Судоку-сайты решают задачу "сыграть один раз". SUDŌ решает задачу **"вернуться завтра"** через:

- Daily Challenge (ежедневный якорь)
- Лидерборд с городами (социальный элемент)
- AI Coach (обучение, а не просто развлечение)
- Прогресс и статистика (ощущение роста)

Это минимальный рабочий прототип продукта, который при дальнейшей разработке может стать полноценным сервисом с подпиской.
